# Lizenzen und Nutzungsgrenzen

Essentia Studio kombiniert Software und Modelle mit unterschiedlichen Lizenzen.
Diese Übersicht ist keine Rechtsberatung.

| Bestandteil | Lizenz | Bedeutung |
| --- | --- | --- |
| Essentia-Studio-Code | MIT | Nutzung, Änderung und Weitergabe gemäß `LICENSE` |
| importierter Navidrome-Smart-Playlist-Katalog und -Logik | MIT | Ursprungsnachweis in `vendor/` beibehalten |
| Essentia / `essentia-tensorflow` | AGPL-3.0 | Lizenz- und Quellcodepflichten der AGPL beachten |
| vortrainierte MTG-Modelle | CC BY-NC-ND 4.0 | Namensnennung; nur nichtkommerziell; keine bearbeiteten Modelle weitergeben |

Insbesondere macht die MIT-Lizenz der Oberfläche die mitgelieferten Modelle nicht
kommerziell nutzbar. Wer Essentia Studio geschäftlich einsetzen möchte, muss vorab
geeignete Modellrechte beschaffen oder eigene kompatible Modelle integrieren.
